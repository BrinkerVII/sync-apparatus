Install via roblox

It's reccomended to install the studio plugin via the ROBLOX website/ROBLOX studio.

Install plugin locally

The plugin on the ROBLOX website might not always be up to date with the plugin from the github release. Installing the plugin locally might conflight with the plugin installed from the roblox website, but it will probably be more up to date. One click of the button does the trick.

Remove locally installed plugin

Removes the plugin you installed with the panel above. If you haven't installed the plugin, this will do absolutely nothing
